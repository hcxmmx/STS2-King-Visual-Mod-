# 👑 Slay the Spire 2 - The Beheaded / King (枭首者视觉重构 Mod)

这是一个面向《杀戮尖塔 2》的高质量角色视觉替换 Mod。核心目标是将储君（REGENT）的整套视觉体验全链路升级为《死亡细胞》中的**“国王 + 夜歌”**主题风格。

我们不仅替换了模型，更重构了角色的动作、特效与交互逻辑，让这种风格从选人界面一路无缝延续到战斗、商店、篝火与顶部栏头像，打造极其沉浸的一体化表现。

## ✨ 核心亮点 (Features)

* **全场景视觉接管**：涵盖选人界面、战斗实体、商店驻留、篝火休息及顶部栏头像的极其完整的模型与 UI 替换。
* **动态双皮肤系统**：选人界面支持**右键点击头像**即时切换皮肤（无伤者白王 / 默认蓝王），并伴随动态弹幕反馈，配置自动本地持久化保存。
* **动作与特效连携 (Combat & VFX)**：拦截底层动画触发信号，为攻击、受击、施法注入专属的武器连段池（如武士刀、平底锅、巨镰等），并与自定义光影 Shader 完美同步。
* **夜歌联动系统 (Serenade System)**：
    * **卡牌响应**：识别特定卡牌（如“君王之剑”）触发夜歌轮盘连击与残影视觉。
    * **形态切换**：从肩部漂浮的“幼年期”到独立飞剑的“完全体”平滑过渡。
    * **专属弹幕引擎 (DanmakuEngine)**：根据战斗情境（受击、铸造、待机等），夜歌会自动触发极其符合原版傲娇人设的文字弹幕演出。

## 🛠️ 技术架构 (Technical Implementation)

本 Mod 采用 Harmony Patch 核心，对角色生命周期进行分层深度接管：
1.  **全局治理**：实现 `KingGlobals` 状态管理，包含资源预载缓存、`KingCombatNodeCache` 节点缓存、失效节点安全清理以及按需求值的日志系统，极大保障了运行帧率与稳定性。
2.  **UI 劫持**：后台静默预加载替换资源，拦截并覆盖原版选人背景、文案及语音锚点。
3.  **渲染层改造**：注入自定义着色器（如无辉光环境下的软光晕 Shader），动态匹配皮肤调色板与发光色阶。

## ⚠️ 已知限制与兼容性 (Compatibility)

* **专属目标**：本 Mod 当前仅对 **REGENT（储君）** 生效，不影响其他角色。
* **潜在冲突**：若其他第三方 Mod 同样深度修改了角色选择界面、顶部栏头像逻辑或战斗动画触发链（`NCreature.SetAnimationTrigger`），可能会由于 Harmony 补丁顺序产生冲突。
* **版本依赖**：部分表现强依赖游戏当前版本的特定节点命名与场景结构（如头像替换逻辑、选人静音恢复等），若官方大版本更新更改了节点树，可能导致局部功能失效。
